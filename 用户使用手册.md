## 快速开始

BananaMall 是一个 AI 驱动的电商详情页生成工具，只需上传产品图片，即可自动生成专业的商品文案和营销图片。

### 核心优势

- 📸 **智能分析**：AI 自动识别产品特征
- ✍️ **文案生成**：一键生成商品标题、描述、规格
- 🎨 **图片创作**：批量生成主图和详情页图片
- 📱 **实时预览**：内置手机模拟器，所见即所得
- 🌐 **多平台**：支持 Amazon、淘宝、京东风格

## 操作流程

完整的工作流程如下：

```
上传图片 → AI分析 → 配置参数 → 生成内容 → 编辑调整 → 导出使用
```

### 第一步：上传产品图片

1. 进入首页，点击上传区域或拖拽图片
2. 选择产品的白底图（建议分辨率 1000x1000 以上）
3. 等待 AI 分析完成，查看产品分析结果

### 第二步：配置生成参数

在配置页面设置以下参数：

| 参数 | 说明 | 建议 |
|------|------|------|
| 目标平台 | 选择电商平台 | 根据实际销售平台选择 |
| 风格 | 设计风格 | 根据品牌调性选择 |
| 语言 | 文案语言 | 根据目标市场选择 |
| 模型 | AI 模型 | 快速用 nanobanana，高质量用 nanabanana |
| 主图数量 | 生成主图张数 | 建议 3-5 张 |
| 详情图数量 | 生成详情图张数 | 建议 2-4 张 |
| 品牌名称 | （可选）品牌名称 | 有品牌可填写 |
| 补充信息 | （可选）产品补充说明 | 用于补充规格等细节 |

### 第三步：生成内容

点击"开始生成"按钮，系统将自动：

1. 生成商品标题、描述、规格列表
2. 生成指定数量的主图
3. 生成指定数量的详情页图片
4. 生成完整的电商详情页内容（5 大模块）

### 第四步：编辑调整

在编辑页面可以：

- **重新生成图片**：点击图片旁边的重新生成按钮
- **编辑图片**：输入提示词对图片进行局部修改
- **修改文案**：直接编辑标题、描述、规格等文字内容
- **预览效果**：使用手机模拟器实时预览页面效果
- **切换风格**：随时切换不同的平台和风格

### 第五步：导出使用

完成编辑后：

1. 点击"导出图片"保存所有生成的图片
2. 点击"导出文案"复制文案内容
3. 将素材上传到对应电商平台使用

## 页面说明

### 首页（上传页）

**功能**：上传产品图片并获取 AI 分析

**操作**：
- 点击上传区域选择图片
- 或直接拖拽图片到上传框
- 查看产品类别、描述、规格等分析结果

**右上角功能按钮**：
- 📜 **历史记录**：查看之前的生成记录
- ⚙️ **设置**：配置 API 和偏好设置

### 配置页

**功能**：设置生成参数

**必填项**：
- 目标平台
- 风格

**可选项**：
- 语言
- 模型选择
- 主图数量（默认 5 张）
- 详情图数量（默认 2 张）
- 品牌名称
- 产品补充信息

### 生成中页

**功能**：显示生成进度

**状态**：
- 文案生成中
- 主图生成中（显示进度）
- 详情页图片生成中

### 编辑页

**功能**：编辑和预览生成的内容

**编辑区**：
- **主图区**：查看、重新生成、编辑主图
- **详情图区**：查看、重新生成、编辑详情图
- **文案区**：修改标题、描述、规格
- **详情页区**：查看 5 大模块内容，可切换不同模块

**预览区**：
- 手机模拟器实时预览
- 支持切换不同模块预览

**导出按钮**：
- 导出图片：下载所有生成的图片
- 导出文案：复制文案到剪贴板

### 设置页

**API 配置**：
- **API Key**：必需，从[飞书文档](https://kege-aigc.feishu.cn/docx/WDXYdMKwFoQrbPxfp7Lcg9OPnBe)获取
- **Base URL**：API 服务器地址（默认：https://api.kegeai.top）

**偏好设置**：
- 默认平台：下次使用时自动选择
- 默认风格：下次使用时自动选择

### 历史记录页

**功能**：查看和管理历史生成记录

**操作**：
- 查看所有历史记录（最多 20 条）
- 点击记录可重新进入编辑页面
- 返回按钮回到首页

## 设置配置

### API Key 配置

1. 点击右上角⚙️设置按钮
2. 输入你的 NanoBanana API Key
3. 保存设置

> **获取 API Key**：点击"没有 API KEY【密钥】？点击注册"链接，按照飞书文档指引获取。

### 自定义 API 地址

如需使用自定义代理，修改 Base URL 即可。

### 默认偏好设置

设置默认平台和风格后，下次使用会自动选中上次的配置，提高效率。

## 常见问题

### Q: 没有图片可以使用吗？

A: 本工具需要上传产品图片进行 AI 分析，无法无图生成。

### Q: 上传的图片有什么要求？

A: 建议使用白底产品图，分辨率至少 1000x1000，图片清晰度越高分析越准确。

### Q: 生成的图片不满意怎么办？

A: 可以点击重新生成按钮，或者使用编辑功能输入提示词进行局部修改。

### Q: 支持批量处理吗？

A: 目前一次处理一个产品，未来计划支持批量功能。

### Q: 文案可以修改吗？

A: 可以，编辑页面所有文案都可以直接修改。

### Q: 如何删除历史记录？

A: 目前历史记录会自动保留最近 20 条，暂不支持手动删除。

### Q: API Key 安全吗？

A: API Key 只存储在本地，不会上传到任何第三方服务器。
